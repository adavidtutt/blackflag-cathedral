# Architecture

The final system is **The Cathedral**.

It does not assume intelligence lives mainly in one dense blob of weights. It distributes cognition across memory species, layered graphs, specialist model species, deterministic stores, retrieval, routing, and consolidation.

The Coding Foundry is one bounded coding lane inside that future system.
