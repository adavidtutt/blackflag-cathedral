# Fractal Cognitive Substrate Tests

This surface holds validation and test-entry planning for the Fractal Cognitive Substrate as a composite Cathedral subsystem.

Use it for:
- cross-layer validation rules
- future structural checks for FCS manifests and doctrine
- test plans for MapFlow legality, layer invariants, and coupling boundaries

Do not use it for:
- implementation-specific runtime tests that belong to a downstream codebase
- freeform brainstorming without a validation target
