# Fractal Cognitive Substrate Layer Registry

This registry tracks the current editable layer surfaces for the Fractal Cognitive Substrate.

Each child subsystem is a coherent insertion point for later passes.

Layer rules:
- `ontology` owns universal node, artifact, payload, and contract law
- `memory-topology` owns plane structure and residency law
- `graph-topology` owns typed graph families, contradiction, and confidence
- `runtime-routing` owns hot route-sketch execution and hydration
- `compiler-learning` owns exhaust capture, rewrite, and promotion loops
- `model-interface` owns dense-model boundaries and translator law
- `mapflow` owns language, IR, legality, and lowering surfaces

Do not add a new layer unless an existing layer cannot own the doctrine without creating ambiguity or contradiction.
