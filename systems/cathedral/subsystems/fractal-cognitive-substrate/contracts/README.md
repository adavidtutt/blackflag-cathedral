# Fractal Cognitive Substrate Contracts

This surface holds contract-law notes for the Fractal Cognitive Substrate as a composite Cathedral subsystem.

Use it for:
- substrate-wide invariants that cross FCS layers
- allowed and forbidden couplings between FCS child subsystems
- contract surfaces that MapFlow or later compiler passes must preserve

Do not use it for:
- general Cathedral doctrine that belongs in the canonical deep schema
- runtime implementation details
- ad hoc notes without contract force
