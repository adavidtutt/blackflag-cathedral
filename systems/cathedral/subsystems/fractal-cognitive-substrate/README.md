# Fractal Cognitive Substrate

This subsystem is the canonical insertion surface for the map-first substrate doctrine inside the Cathedral.

Use it to organize future passes by layer rather than by thread order.

Extension rules:
- add doctrine to the smallest coherent FCS layer that can own it
- update the deep schema first when the addition changes cross-layer law
- keep root-level additions limited to cross-layer invariants and shared doctrine
- do not add free-floating concepts here if a child layer can own them cleanly
